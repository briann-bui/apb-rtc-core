APB RTC Core

A simple binary-calendar RTC with a 32-bit APB interface, programmable prescaler, calendar rollover, alarm matching, and interrupts.

Block Flow

APB registers → prescaler → time/calendar counters → alarm comparator → IRQ

Register Map

AddressNameDescription
0x00CTRLRTC and alarm control
0x04STATUSRunning and pending status
0x08PRESCALERPCLK cycles minus one per second
0x0CTIMEBinary hour, minute, second
0x10DATEBinary year, month, day
0x14ALARM_TIMEAlarm time
0x18ALARM_DATEOptional alarm date
0x1CIRQ_ENInterrupt enables
0x20IRQ_STATWrite-one-to-clear pending bits
0x24VERSIONIP version

Verification

Run make lint for RTL lint and make run for the UVM regression.
